## Overview

The **IT Problems & Instant Diagnosis Table** is a lightweight interactive reference tool for IT support professionals. It provides:

* A **searchable and filterable table** of common IT issues
* Categorized symptoms, diagnosis steps, and quick resolution guidance
* A clean browser interface that expands rows to show detailed troubleshooting steps

This table consolidates over *a hundred real-world IT issues* across hardware, network, software, cloud, and workplace systems into a single instantly accessible page. ([GitHub](https://raw.githubusercontent.com/tuleenitech/IT-Diagnosis-Table/main/index.html "IT Problems & Instant Diagnosis Table"))

## Getting Started

### Clone the Repository

```sh
git clone https://github.com/tuleenitech/IT-Diagnosis-Table.git
cd IT-Diagnosis-Table
```

### Open in Browser

You can open the `index.html` file directly in your browser to use the tool locally:

```sh
open index.html
```

Or use a simple static server:

```sh
npx serve .
```
